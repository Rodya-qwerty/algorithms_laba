# Данный модуль реализует бинарное дерево поиска (BST) для хранения целых чисел
Функции
---
 - insert
 - inorder
 - search

## insert(root, value)
Назначение: Вставляет новое значение в бинарное дерево поиска с сохранением свойства упорядоченности.
### Принцып работы:
- Если дерево пустое (root is None) → создаётся новый узел, который становится корнем
- Если value < root.value → рекурсивная вставка в левое поддерево
- Если value > root.value → рекурсивная вставка в правое поддерево
- Если value == root.value → значение игнорируется (дубликаты не добавляются)

## search(root, target)
Назначение: Выполняет поиск значения в дереве и возвращает узел, содержащий его.
### Принцып работы:
- Если корень пустой (None) или значение корня равно искомому → возвращаем корень
- Если искомое значение меньше текущего → идём в левое поддерево
- Если искомое значение больше текущего → идём в правое поддерево

## inorder(root, result=None)
Назначение: Выполняет симметричный обход (инфиксный обход) дерева и возвращает отсортированный список всех значений.
### Принцып работы:
1. Если узел существует:
   - a. Рекурсивно обойти левое поддерево
   - b. Добавить значение текущего узла в результат
   - c. Рекурсивно обойти правое поддерево
2. Вернуть накопленный результат
