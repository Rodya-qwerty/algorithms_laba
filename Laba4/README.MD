Данный проект представляет собой консольное приложение для ведения электронного журнала через sqlite. 

## Функциональные возможности
    1. Управление пользователями
     -Регистрация новых преподавателей
     -Авторизация существующих пользователей
     ( при создании объекта администратор по умолчанию (login: admin, password: admin) )

    2. Работа со студентами
     -Добавление учеников в журнал (имя, фамилия, класс)
     -Поиск учеников по ID, имени или фамилии
     -Просмотр всех учеников класса, отосительно преподователя 

    3. Работа с оценками
     -Выставление оценок (от 2 до 5)
     -Просмотр всех оценок ученика 

    4. Просмотр успеваемости
     -Вывод всех студентов с их оценками
     -Отображение среднего балла по каждому студенту
    
**<h2>Устройство БД представленно в прикреплённой фотографии</h2>**

## Технологии
1. Язык программирования: Python 3
2. Баpа данных: SQLite3
3. Хеширование паролей: SHA-256
4. Ввод пароля: Модуль getpass 

## Безопасность
- Пароли хранятся в базе данных в захешированном виде (SHA-256)
- При вводе пароля символы не отображаются на экране
- Проверка существования пользователя при регистрации

## Файлы проекта
```
table.py - основной код с классом Journal
main.py - точка входа в программу
journal-school.db - файл базы данных (создаётся автоматически)
```

# Примеры использования

Регистрация нового преподавателя:
```text
    ВХОД В СИСТЕМУ
    1. Войти
    2. Зарегистрироваться
    3. Выйти

Выберите действие: 2

РЕГИСТРАЦИЯ ПРЕПОДАВАТЕЛЯ
Придумайте логин: ivanov
Придумайте пароль: ****
Подтвердите пароль: ****
Преподаватель 'ivanov' успешно зарегистрирован!
```
Добавление студента:
```text
ДОБАВЛЕНИЕ СТУДЕНТА
Введите имя студента: Иван
Введите фамилию студента: Петров
Введите номер класса: 10
Введите букву класса: А

Студент успешно добавлен!
   ID: 1
   Имя: Иван
   Фамилия: Петров
   Класс: 10А

```
Выставление оценки:
```text
Введите ID или имя студента: Иван
Введите оценку для Петров Иван (2-5): 5
Оценка 5 успешно добавлена!
```
Просмотр журнала:
```text
ЭЛЕКТРОННЫЙ ЖУРНАЛ
ID  Фамилия         Имя          Класс  Оценки       Ср.
1   Петров          Иван         10А    5, 4, 5      4.67
2   Сидоров         Алексей      10А    3, 4         3.5
```
